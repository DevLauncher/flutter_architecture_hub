# 项目文档

🌐 [English](README.md) | [Türkçe](README-TR.md)

此项目遵循 Clean Architecture 原则并使用 Flutter。

## 文件结构
- **lib/app/**: 全局配置、路由和根组件。
- **lib/core/**: 公共提供程序、帮助程序和常量。
- **lib/features/**: 按层划分的功能模块。
- **lib/main.dart**: 应用程序入口点。
